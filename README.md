# BloodLink_DB
BloodLink is a web-based Blood Bank and Donor Management System that provides a centralized platform for registering donors, managing blood inventory across multiple blood banks, and handling hospital blood requests.
Donors can register, check their eligibility, and view their donation history. Blood bank
administrators can manage stock levels and receive automatic alerts when inventory for a
specific blood type falls below a threshold. Hospitals can submit blood requests specifying the
required type and quantity, and the system matches available inventory from registered blood
banks. The system improves response time in emergencies by eliminating the need for manual
phone-based coordination and centralizing all blood availability data in one place.
